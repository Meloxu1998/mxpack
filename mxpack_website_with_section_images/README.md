# MX PACK Filled Website

This package is based on the user-provided Tailwind single-page website structure and filled with MX PACK / custom packaging manufacturer content.

## Deploy to Cloudflare Pages

- Push all files to a GitHub repository.
- In Cloudflare Pages, connect the repository.
- Framework preset: None.
- Build command: leave empty.
- Output directory: `/`.

## Replace before launch

- Phone number: `+86 757 0000 0000`
- Email/contact form target
- Real product photos in `assets/img/`
- Final logo file
- Domain in sitemap/robots if you add them later
